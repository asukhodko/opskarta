# Пример: Hello (базовый план с представлениями)

Этот пример демонстрирует базовое использование формата opskarta с файлом плана и файлом представлений.

## Что демонстрирует

- **Файл плана** (`hello.plan.yaml`) — описание проекта с узлами, статусами и зависимостями
- **Файл представлений** (`hello.views.yaml`) — определение Gantt-представления для визуализации плана

## Файлы

| Файл | Описание |
|------|----------|
| `hello.plan.yaml` | План проекта "Обновление Git-сервиса" с фазами и задачами |
| `hello.views.yaml` | Определение Gantt-представления с одной дорожкой |

## Структура плана

План описывает простой проект обновления сервиса:

```
root (Обновление Git-сервиса)
├── prep (Подготовка) — 10 дней
├── rollout (Раскатка) — 5 дней, после prep
└── switch (Переключение трафика) — 1 день, после rollout
```

## Использование

### Валидация

```bash
cd specs/v1
python tools/validate.py examples/hello/hello.plan.yaml examples/hello/hello.views.yaml
```

### Генерация Mermaid Gantt

```bash
cd specs/v1
python -m tools.render.mermaid_gantt \
  --plan examples/hello/hello.plan.yaml \
  --views examples/hello/hello.views.yaml \
  --view overview
```

## Особенности примера

- Использует кастомные статусы с цветами
- Демонстрирует зависимости между узлами (`after`)
- Показывает использование `duration` и `start` для планирования
- Включает заметки (`notes`) для критичных задач
